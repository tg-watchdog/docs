# Version 2207.01

## Release Date
July 6th, 2022

## Update in Bot Program
- Web app now supports different locales. (This feature is in experiencing.)
  - Now supports English, Chinese, Simplified Chinese (China Mainland), Traditional Chinese (Taiwan), Traditional Chinese (Hong Kong)
- New locales supported by chatting interface: Traditional Chinese, Traditional Chinese (Hong Kong), Japanese. (This feature is in experiencing.)

## Known Issues
- Chatting interface or web app may use English which is fallback option, even use the OS or Telegram app with supported languages.
- Chatting interface or web app may use English which is fallback option when you are using third-party language packages.
